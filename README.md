# Class Assistant Bot

A smart classroom assistant powered by Mistral AI.

## Features
- Answers student questions in real-time
- Helps with homework and explanations
- Remembers class context using local database
- Powered by Mistral (Codestral, Mistral Large, etc.)

## Setup (DisCloud)
1. Upload this folder to DisCloud
2. Rename `.env.example` → `.env`
3. Fill in your tokens and API keys
4. Deploy!

The bot starts from `main.py` automatically.
